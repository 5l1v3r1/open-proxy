# open-proxy
一键部署被墙网站反向代理，在无翻墙工具的情况下便捷地获取被封锁的国际新闻及国内禁闻

##### 鉴于GFW封锁不断升级, 强烈推荐使用 [一键翻墙软件](https://github.com/gfw-breaker/nogfw/blob/master/README.md) 或 使用 [禁闻聚合](https://github.com/gfw-breaker/banned-news/blob/master/README.md)
